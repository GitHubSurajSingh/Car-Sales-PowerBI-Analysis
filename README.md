# Indian Automobile Sales Analytics Dashboard

## Overview

This project presents an interactive Power BI dashboard created using a dataset of 10,000 automobile dealership sales transactions across India.
The goal of the project is to analyze revenue performance, profitability, customer behavior, and regional demand patterns.

## Tools Used

Power BI – Data visualization, data modeling, DAX measures
Python – Dataset generation using Faker library
Excel/CSV – Data storage and preprocessing

## Key Features

• KPI tracking (Revenue, Profit, Units Sold, Average Selling Price)
• Brand performance comparison
• Fuel type and transmission preference analysis
• Geographic sales distribution across cities
• Monthly sales trend and seasonality analysis

## Business Value

The dashboard helps automobile dealerships understand:

* which car brands generate the most revenue
* which customer segments are most profitable
* regional sales hotspots
* demand trends for fuel types and vehicle segments
